# Expensee Landing Page

The public-facing landing page for **Expensee** — private, real-time Web3 payroll built on Solana.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Sections

- **Hero** — Headline with holographic Agentic Stream visual
- **Partner Logos** — Solana, USDC, Inco, Magicblock
- **Benefits** — Core advantages (FHE privacy, real-time accrual, stealth addresses)
- **Global Scale** — Live intelligence dashboard
- **Social Proof** — Testimonials
- **FAQ** — Common questions
- **Footer** — Links and socials

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Three.js / React Three Fiber
- TypeScript
